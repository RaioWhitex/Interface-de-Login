# 🔐 Login Form UI

Um projeto simples e responsivo de **Formulário de Login**, desenvolvido para praticar conceitos de **HTML5** e **CSS3**, com foco em usabilidade, organização visual e experiência do usuário.

---

## 📸 Preview do Projeto

Interface contendo:

- Campo para e-mail ou telefone
- Campo para senha
- Botão de login
- Link para recuperação de senha
- Link para cadastro de novos usuários
- Ícones ilustrativos nos campos

---

## ✨ Tecnologias Utilizadas

- **HTML5**
- **CSS3**
- **Font Awesome**

---

## 🎨 Características do Design

- Layout moderno e minimalista
- Interface limpa e intuitiva
- Paleta de cores suaves
- Campos de entrada estilizados
- Botão de ação destacado
- Estrutura responsiva

---

## 🚀 Objetivo do Projeto

Praticar habilidades de desenvolvimento front-end através da criação de uma interface de autenticação moderna, explorando conceitos de formulários, estilização e responsividade.
